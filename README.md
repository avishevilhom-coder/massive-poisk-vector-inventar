Система инвентаря и использования предметов (C++)

Описание
Консольная программа на C++, демонстрирующая:
создание инвентаря через vector
поиск предметов с помощью цикла for
применение эффектов (лечение, увеличение урона)
удаление предметов после использования
обработку случаев, когда предметов нет
Технологии
C++ (стандарт 17)
STL: vector, string
 Работа с консолью (Windows)

Как это работает
1. Создаётся инвентарь с двумя предметами:
Зелье здоровья (+20 HP)
Зелье урона (+10 урона)
2. Пользователь выбирает предмет (1 или 2)
3. Программа ищет предмет в инвентаре через цикл for
4. Эффект применяется к игроку
5. Предмет удаляется из инвентаря
6. Цикл повторяется, пока предметы не закончатся
